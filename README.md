# AutoRep AI — Door Step Auto

**Version 1.1.0-rc.1** (See [CHANGELOG.md](./CHANGELOG.md))
*Premium AI-powered automotive sales agent platform with Apple-grade PWA mobile support.*

AutoRep AI handles customer conversations across phone, SMS, website chat, Facebook, and Instagram — guiding shoppers from first contact through vehicle match, quote, appointment, and finance intake.

## Product Surfaces

1. **Live Agent Console** — Flagship real-time conversation workspace with inbound queue, customer snapshot, deal stage, inventory matches, and handoff controls
2. **Omnichannel Conversations** — Unified thread model across calls, SMS, chat, and social DMs with delivery state, opt-out controls, and AI disclosure
3. **Leads & Follow-Up** — Lead inbox by source, priority, stage, and follow-up status with CRM sync and duplicate detection
4. **Vehicles & Quotes** — Inventory search with quote builder, scenario comparison, and revision history
5. **Finance Intake & Routing** — Secure customer intake, document checklist, consent tracking, and Dealertrack routing
6. **Manager Supervision** — Live conversation monitoring, sentiment analysis, escalation alerts, and human handoff controls
7. **Integrations & Configuration** — Telephony, SMS, Meta, CRM/DMS, AI model, and webhook setup

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS with custom design tokens (obsidian black, ivory, gold)
- Supabase (database, auth, edge functions)
- shadcn/ui component library

## Getting Started

The app is fully built and ready for API key configuration. Connect your external services through the Integrations page.
